# Shohail Ahmed Portfolio

Static portfolio webpage.

## Open locally

```bash
python3 -m http.server 4174 --directory portfolio
```

Then open `http://localhost:4174`.

## Files

- `index.html`: page content, metadata, JSON-LD.
- `styles.css`: light/dark visual system and responsive layout.
- `script.js`: current-year and theme-toggle enhancement.
- `assets/`: resume and proof imagery.

## Pre-publish checks

- Confirm LinkedIn URL if you want a bot-verifiable check; LinkedIn may block automated requests.
- Add HackerRank rank proof if available.
- Add public AI-Worker ownership proof if the "co-owner" wording should be independently verifiable.
